Use of Prompt-Based Learning for Code-Mixed and Code-Switched Text Classification

Pasindu Udawatta, Indunil Udayangana, Chathulanka Gamage, Ravi Shekhar, Surangika Ranathunga
World Wide Web • Volume 27, Article Number 63

Explores code-mixed and code-switched text classification using prompt-based learning. It introduces a novel approach which combines script-aware modeling with task-specific adapters. Experiments show that this method outperforms standard fine-tuning and prompting techniques across sentiment analysis, hate-speech detection, and humour classification.

TTSVowelViz: A Tool for Visualising Text-to-Speech Model Training Through Vowel Spaces

Pasindu Udawatta, Jesin James, B.T. Balamurali, Catherine Watson, Ake Nicholas, Binu Abeysinghe
ICASSP 2025 - Show & Tell • IEEE International Conference on Acoustics, Speech, and Signal Processing

TTSVowelViz is an interactive tool that visualises vowel spaces in real-time during text-to-speech model training. By mapping vowel sounds using formant frequencies, it helps to identify which sounds are accurately learned, enabling early detection of issues without costly retraining. It bridges the gap between linguistic analysis and model development.
